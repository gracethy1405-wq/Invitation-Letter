## How to customize `index.html` (search for these anchors)
- **Wording / emojis:** edit the text in the `<body>` (hero section + the `.card`).
- **His name:** add it in the hero `<h1>` / `.title` (e.g. "hey [name] 🥺").
- **Date & time:** by default it auto-targets the **upcoming Friday 7pm** so the
  countdown, the "when" chip and the calendar link all agree. To pin a real date,
  find `nextFriday7pm()` and replace `var dateObj = nextFriday7pm();` with e.g.
  `var dateObj = new Date(2026, 6, 17, 19, 30);` (month is 0-based → 6 = July).
- **Where / vibe:** the `.meta` chips and `EVENT.location`.
- **Colors:** the `:root { ... }` block at the top of `<style>` (purple palette).
- **The "no" button:** `var noSeq = [...]` controls the loop
  (`no → noo → nooo → noooo → please no 🥺`, then repeats).
- **Where suggestions go:** `var FEEDBACK_TO = "gracethy1405@gmail.com";`

## The Tzuyu pics
Drop images into `pics/` named `beg1.jpg`, `beg2.jpg`, `beg3.jpg`, then
`git add pics && git commit -m "add pics" && git push`. Until added, the three
slots show 🥺 / 🙏 / 😭 placeholders. (Repo is public — only use pics you're ok
sharing publicly.)

## The "suggest changes" box
- It's the always-visible box at the bottom of the card.
- **send it to me 💌** opens the recipient's email pre-filled to `FEEDBACK_TO`;
  **copy** copies their note to paste anywhere.
- ⚠️ The email is visible in the public page source (mild spam risk). Alternatives
  if you care: a free form service (Formspree / Google Form), or switch the whole
  document to a **Google Doc** shared as "Commenter" so they can comment *inside*
  the doc instead of beside it (then point the doc link/PDF chip at the Doc URL).

## Regenerating the PDF
`invitation.pdf` was generated by a small Python script (hand-built one-page PDF,
purple title). There's no script file saved in the repo — easiest is to just edit
the text and ask Claude to regenerate it, or replace `invitation.pdf` with any PDF
of your own (keep the filename, or update `DOC_URL` in `index.html`).
