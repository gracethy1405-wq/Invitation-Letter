Drop your Tzuyu "begging" pictures in THIS folder, named exactly:

  beg1.jpg
  beg2.jpg
  beg3.jpg

(.jpg, .jpeg or .png all work as long as the name matches — if you use .png,
change the src in index.html from beg1.jpg to beg1.png, etc.)

They'll automatically show in the three picture slots on the invitation.
Until you add them, the slots show big 🥺 / 🙏 / 😭 faces instead, so the page
still looks intentional.

To publish them to the live link after adding the files:
  git add pics
  git commit -m "Add begging pics"
  git push

Note: this repo is PUBLIC, so anything you put here is visible to anyone with
the link. Use pictures you're comfortable sharing publicly.
